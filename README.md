## 🌸 Overview
**CaraQAway** is a calm, creative QA portfolio that blends testing principles, motivation, and project-based learning.  
It represents my approach to software quality — thoughtful, structured, and user-focused — and documents my path into tech through continuous improvement and curiosity.

> *“QA is how we keep promises to users.”*

---

## 💡 What You’ll Find

### 💬 Core Sections
- **About QA:** Turning “it should work” into “it does work.”  
- **Principles I Follow:** My foundation for reliable, repeatable QA.  
- **30 Testing Types:** A visual summary of QA test categories.  
- **Externship:** My 7-week automation project with *Tripleten × Runestone Academy*.  
- **Bug Jam Projects:** Real QA debugging challenges in collaborative sprints.  
- **About Me:** My story — from landscaping to leading test automation and QA excellence.

## 🧪 Testing Philosophy
**Shift-Left Testing** — Plan early and prevent issues before merge.  
**Exploratory Testing** — Find what automation can miss.  
**Observability** — Logs and metrics accelerate debugging.  
**Consistency** — Small, daily progress compounds into quality.
